## What is Halo?

Halo is a powerful and easy-to-use open-source website building tool. From personal blogs and knowledge bases to corporate websites and online stores, Halo helps you build it all — meeting your diverse website needs in one place.

## Quick Start

If you have Docker installed, use the following command to quickly start a Halo demo environment:

```bash
docker run -d --name halo -p 8090:8090 -v ~/.halo2:/root/.halo2 halohub/halo:2.24
```

Or click the buttons below to launch via [Gitpod](https://gitpod.io/) or [ClawCloud Run](https://template.us-west-1.run.claw.cloud/deploy?templateName=halo):

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/#https://github.com/halo-sigs/gitpod-demo)

[![Run on ClawCloud](https://raw.githubusercontent.com/ClawCloud/Run-Template/refs/heads/main/Run-on-ClawCloud.svg)](https://template.us-west-1.run.claw.cloud/deploy?templateName=halo)

**The above methods are for evaluation only.** For production, we recommend deploying via [1Panel](https://github.com/1Panel-dev/1Panel), a Linux server management panel ([see docs](https://docs.halo.run/getting-started/install/1panel)) for easy reverse proxy, SSL certificates, and backup management. For more deployment options, [see the documentation](https://docs.halo.run/category/%E5%AE%89%E8%A3%85%E6%8C%87%E5%8D%97).

## Edition Comparison

**Halo Community Edition**: Open-source and free, licensed under GPLv3.

- [x] For individual developers, tech enthusiasts, and open-source projects
- [x] Zero-cost setup for blogs, portfolios, and technical documentation sites
- [x] Over 100 free themes and plugins

**Halo Pro Edition**: Adds 10+ premium features on top of Community Edition.

- [x] Mobile app: manage content anywhere
- [x] AI-powered site building: generate professional sites quickly
- [x] Phone number verification login: improved security and UX
- [x] Full-site private deployment: data sovereignty
- [x] Paid theme/plugin marketplace: premium themes, SEO optimization, paid reading, AI assistant, and 10 paid plugins

**Halo Business Edition**: Adds a full-featured online store on top of Pro Edition.

- [x] Integrated online store: product management, order processing, payment integration
- [x] Built for Chinese merchants: seamless WeChat Pay, Alipay, and local ecosystem
- [x] Brand website + CMS + online store in one — drive business growth

For a detailed comparison, see [Edition Comparison](https://www.lxware.cn/halo).
