# Nekwasar Blog

Forked from [halo-dev/halo](https://github.com/halo-dev/halo) v2.24.
Custom theme: **Anon** — glassmorphism, rainbow borders, dark/light toggle, anonymous commenting.

**Design system:** Nebula Glass — backdrop-filter blur, conic-gradient borders (dark mode), holographic sheen, particle grid, staggered scroll reveals, ripple clicks.

**Routes:** `/` `/archives` `/archives/{slug}` `/categories` `/categories/{slug}` `/tags` `/tags/{slug}` `/links` `/moments` `/photos`

**Stack:** Java 21, Spring Boot WebFlux, Vue 3, TypeScript, Thymeleaf, Alpine.js — 28 custom templates, Lucide icons.

**Deploy:** `docker run -d --name halo -p 8090:8090 halohub/halo:2.24` then upload `theme-anon.zip` via Console > Themes.
